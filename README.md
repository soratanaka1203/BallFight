# BallFight
## 【作品名】
BallFight
githubのURL：https://github.com/soratanaka1203/BallFight

## 【制作期間】
2024年12月 ～ 2025年2月（約3か月）

## 【開発人数】
1人（個人制作）

## 【開発環境】
Unity 2022.3.36f1
Visual Studio 2022
Windows 11

## 【ジャンル】
対戦ゲーム

## 【概要】
プレイヤーを操作して対戦するアクションゲームです。
スキルや体当たりを使い、他プレイヤーを落とすことを目指します。
生き残った最後のプレイヤーが勝利します。


## 【操作方法】
### キーボード							
W / A / S / D	：移動/カーソル移動/キャラクター選択<br>
Space		：ジャンプ<br>
Shiftキー	：ダッシュ<br>			
Jキー		：スキル<br>
Enterキー	：決定<br>			
Esc		：ポーズメニュー  

### コントローラ
左スティック	：移動/カーソル移動/キャラクター選択<br>
Aボタン		：ジャンプ/決定<br>
Xボタン		：ダッシュ<br>
Yボタン		：スキル<br>
Startボタン	：ポーズメニュー
